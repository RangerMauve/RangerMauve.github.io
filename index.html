Mauve Signweaver

A Canadian tech enthusiast with a passion for decentralization.

Mauve uses cutting edge tech to create applications and software development kits that can be used by people across different platforms and use cases. Their current focus is on making p2p and mesh technology more accessible to developers so that they can create applications and experiences that preserve user agency and privacy.

A main project for this is Agregore, a web browser that combines different peer to peer protocols together. Another is Distributed Press which makes it easier to publish web content across different peer to peer protocols, which can then be loaded through browsers like Agregore. They have also dipped outside 2D by adding Hypercore support to the Godot game engine in the HyperGodot library which enables fully p2p virtual worlds and games with compatibility with the distributed web.
