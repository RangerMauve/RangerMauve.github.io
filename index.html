RangerMauve

A Canadian tech enthusiast with a passion for decentralization.

Mauve uses the web and Node.js to create applications and software development kits that can be used by people across different platforms and use cases. Their current focus is on making p2p and mesh technology more accessible to developers so that they can create applications and experiences that preserve user agency and privacy. In addition to regular decentralized web tech, Mauve is working on building up a cross-platform local-first cyberspace stack using WebXR and the Dat protocol.
